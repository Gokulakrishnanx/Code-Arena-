# Code Arena - Points Management System

A web-based points management system for managing and displaying team points in a coding competition.

## Features

- Real-time points tracking
- Team management
- Round-wise points entry
- Live points table with automatic updates
- Admin dashboard for points management
- Responsive design for all devices

## Pages

1. **Home Page (index.html)**

   - Welcome screen
   - Navigation to points table and admin dashboard

2. **Points Table (points.html)**

   - Live display of team rankings
   - Round-wise points display
   - Total points calculation
   - Auto-refresh every 5 seconds

3. **Admin Dashboard (admin.html)**
   - Secure login system
   - Team management (add/delete teams)
   - Points entry for each round
   - Real-time total points calculation
   - Points table update functionality

## Setup

2. Open `index.html` in a web browser

- HTML5
- CSS3
- JavaScript (Vanilla)
